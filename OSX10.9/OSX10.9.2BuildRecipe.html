1. Back everything up to another computer
  2. Hold down Command+Option+P+R to reset PRAM (let it reboot twice)
  3. Boot a Linux distro and shred the hard drive: shred -v /dev/sda
  4. Power off, attach external install disk, power on
  5. OS X will boot from external disk (this will take a while>
  6. Press Return to use English as the main language
  7. Select Disk Utility, click Continue
  8. Select the internal hard disk, click Erase (at top)
  9. Ensure Format is Mac OS Extended (Journaled)
  10. Name the disk “HD”
  11. Click Erase (near the bottom right), click Erase on popup
  12. When it finishes, quit Disk Utility
  13. Select Install OS X, click Continue
  14. See the OS X Mavericks screen, click Continue
  15. Click Agree, click Agree on popup
  16. Select HD, click Install
  17. It will display "Preparing to install. Your computer will restart automatically." for several minutes
  18. It will display "about a second remaining" for about an hour
  19. Eventually, it will display Installing OS X on the disk "HD"...
  20. Select United States, click Continue
  21. Select U.S., click Continue
  22. Do not select a network, click Continue, click Continue on popup
  23. Select Don't transfer any information now, click Continue
  24. Click Agree, click Agree on popup
  25. Set administrative account details, click Continue
  26. Set ComputerName: scutil --set ComputerName HOSTNAME
  27. Set HostName: scutil --set HostName HOSTNAME
  28. Set LocalHostName: scutil --set LocalHostName HOSTNAME
  29. Create a non-administrative user account
  30. Shutdown the computer, power off the the external disk, and disconnect it
  31. HWReg
