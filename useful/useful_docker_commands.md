from command line
-----------------

```
docker ps -a

docker images

docker info

docker system prune

docker build .

docker build . -t image_tag_name

docker run --name container_name -d -p 4000:4000 image_tag_name

docker exec -it image_tag_name bash

docker stop container_name

docker compose up
```
