from command line
-----------------

`VBoxManage list vms`

`VBoxManage list runningvms`

`VBoxManage list ostypes`

`VBoxManage showvminfo <vmname>`

`VBoxManage modifyhd ~/VirtualBoxVMs/vm/vm.vdi --resize 10240`
